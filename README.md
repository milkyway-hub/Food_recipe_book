# Food Recipe Book

## Project Overview
The **Food Recipe Book** is a dynamic web application designed to make exploring recipes simple, interactive, and enjoyable. Focusing on **Indian cuisine**, users can browse a variety of recipes, view details, and search for their favorite dishes using an intuitive interface.  

This project demonstrates modern web development techniques and emphasizes modular design, enhancing both **user engagement** and **maintainability**.

---

## Features
- **Recipe Cards:** Display recipes in visually appealing cards for easy browsing.  
- **Search Functionality:** Quickly find recipes by ingredients, cuisine type, or recipe name.  
- **Modal Pop-ups:** Click on a recipe to view detailed instructions, ingredients, and cooking steps in a modal window.  
- **Responsive Design:** Works seamlessly on desktop and mobile devices.  
- **User-Friendly Interface:** Clean and interactive UI for an enjoyable browsing experience.  

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Design Techniques:** Responsive layout, modular components, modal pop-ups
