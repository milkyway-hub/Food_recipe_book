# 🍛 Food Recipe Book – Interactive Recipe Explorer

## 📌 Project Overview
Food Recipe Book is a simple and interactive web application that allows users to explore different food recipes in an organized and visually appealing way. Users can browse recipes, search for dishes, and view detailed cooking instructions through a clean and user-friendly interface.

This project demonstrates the use of **HTML, CSS, and JavaScript** to create a responsive and interactive frontend application.

---

## 🎯 Project Highlights

- Interactive recipe browsing interface  
- Dynamic search functionality  
- Responsive card-based layout  
- Modal popup for detailed recipe information  
- Clean and user-friendly UI  

---

## ✨ Features

- 📖 **Recipe Cards**  
  Recipes are displayed in visually structured cards for easy browsing.

- 🔍 **Search Functionality**  
  Users can quickly search recipes using the search bar.

- 🪟 **Recipe Detail Modal**  
  Clicking on a recipe opens a popup modal displaying:
  - Ingredients
  - Cooking instructions
  - Recipe image

- 🎨 **Interactive UI**  
  Smooth hover effects and interactive components enhance user experience.

- 📱 **Responsive Design**  
  Works smoothly across desktop, tablet, and mobile devices.
